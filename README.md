# Lumora Studio

https://lahieuphong.github.io/lumora-studio/

## Commands

```bash
yarn install          # install dependencies
yarn dev              # start dev server at localhost:5173
yarn build            # build production → dist/
yarn preview          # preview production build locally
```

### Yarn cache

```bash
yarn cache list       # list cached packages
yarn cache clean      # clear all cache
```
